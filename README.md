# RSoC-FPGA-Sound-Generator
Complete FPGA sound generator for RSoC module (822H1). Labs: PWM tone generation, MicroBlaze SoC, sound font ROM with 4 instruments, and dual-port RAM with processor-controlled waveforms. Fully documented with simulation and MATLAB analysis.

**Student:** Denilson Domingos Monteiro Nicolau  
**Module:** Reconfigurable Systems on Chip (822H1) - University of Sussex

## Labs Overview

| Lab | Title | Key Features |
|-----|-------|--------------|
| 3-5 | PWM Tone Generation | 440Hz tone, volume control, MATLAB analysis |
| 6 | MicroBlaze SoC | Softcore processor, GPIO, timer interrupts |
| 8 | Sound Font ROM | 4 instruments, DDS, MATLAB waveform generation |
| 9 | DP-RAM + Processor | Dynamic waveform loading, hardware/software co-design |

## Hardware
- Nexys 4 FPGA (xc7a100tcsg324-3)
- 3.5mm audio jack output
- 16 switches for control

## Tools
- Vivado 2024+
- Xilinx SDK / Vitis
- MATLAB
